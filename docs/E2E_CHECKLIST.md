## 진짜 E2E (전체 사이클 육안)

### E2E-01 — 신규 사용자 첫 주 `python tests/manual_launch.py e2e1`
- [ ] 오늘 탭 = S0 "이번 주 계획이 아직 없어요" 히어로
- [ ] "이번 주 계획 세우기 →" → 이번 주 탭 전환
- [ ] 계획 생성·저장 → "저장 완료!"
- [ ] 오늘 탭 재접속 → S1 전환(세션 히어로 + 카드 자동 생성)
- [ ] "다녀왔어요" → 기록 입력 → "기록 저장하기 →"
- [ ] S3 전환(km bignum + 코치 분석 카드)
- [ ] 이번 주 탭 → 완료 거리/세션 증가
- [ ] 기록 탭 → 오늘 기록 표시

### E2E-02 — 설정 변경 → 계획 갱신 (F12 흐름 포함)
- [ ] E2E-01 끝난 상태에서 설정 탭 → 프로필 변경 → 저장
- [ ] 갱신 다이얼로그 자동 오픈
- [ ] "나중에" → 플래그 삭제, 재오픈 안 됨
- [ ] 다시 변경 → "지금 반영" → 주간 계획 재생성·저장됨

### E2E-03 — 통증 코칭 사이클 `python tests/manual_launch.py e2e3`
- [ ] 기록 입력 시 통증 ON(`왼쪽 발목` / `4/10`) → 저장
- [ ] `05_INJURY_HISTORY.md` 에 오늘 날짜 행 추가(부위·수준·종류 포함)

## 자정 날짜 경계 · 새로고침 상태 유지

### 자정 경계 (토 → 일, 같은 주)
- [ ] `python tests/manual_launch.py sat` → 토요일 상태 확인(롱런 세션 보임)
- [ ] Ctrl-C 종료 → `python tests/manual_launch.py sun` → 일요일 재기동
- [ ] 일요일 = S5 "🏆 이번 주 훈련 종료" 히어로 전환
- [ ] 요일 판정·today_log 가 일요일 기준으로 바뀜
